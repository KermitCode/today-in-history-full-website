# today-in-history-full-website
历史上的今天网-按日期记录着历史上的今天发生的大事。此为2012年使用YII框架完全独立写的一个网站（中英双语版），包括抓取全站12000+历史事件记录，并调用google进行全部翻译，同时有开发完整管理后台。
